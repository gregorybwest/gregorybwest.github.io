Local Gigs App

The culmination of my learnings at Actualize: Local Gigs. A place where concertgoers can find all the music events happening locally on a map, and music artists can search for venues and post their upcoming local shows, as well as have their own profiles where potential fans can hear their music via an embedded Spotify link. The app utilizes a Rails backend framework, a Vue.js frontend, persists the data using PostgreSQL, as well as using multiple API endpoints using the Yelp Fusion API and the Mapbox library. I'm excited to continue to add more features to the project.
